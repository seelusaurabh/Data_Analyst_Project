# Data_Analyst_Project
Data Analyst Project Using MySQL, Python, and Power BI

## 📊 Problem Statement
The project analyzes customer shopping behavior data to identify purchasing patterns and customer trends.  
The goal was to help businesses understand top products, spending habits, and customer demographics.

## 🛠️ Tools Used
- **MySQL** → for data storage and queries
- **Python** → for cleaning and analysis
- **Power BI** → for dashboards and visualization

## 📂 Files in this Repo
- `Data_Analyst_Project.sql` → SQL queries
- `Data_Analyst_Project.ipynb` → Python notebook
- `customer_behaviour_dashboard.pbix` → Power BI dashboard
- `project_report.docx` → Detailed report
- `Customer-Shopping-Behavior-Analysis.pdf` → Final presentation

## 🔑 Key Insights
- Top 3 most purchased products by category
- Spending behavior varies across age groups
- Visualization of purchase frequency and customer trends

## 📌 How to View
- Open the `.pbix` file in Power BI for dashboards
- Read the report (`project_report.docx`) for detailed analysis
- Use the PDF presentation for interview discussions
